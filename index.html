RESPONSIVE

'Coming Soon' is a simple one page template that allows you to keep your users informed while you are building a new website.

Be it a 17" laptop screen or an iPhone. It's based on a responsive grid and media queries. Go ahead, resize this 'coming soon' page to see the grid in action.

Fork, Tweet, Share & be OpenSource.

SKELETON

This Coming Soon free template is based on Skeleton who is built on three core principles:

  • A Responsive Grid Down To Mobile: Elegant scaling from a browser to tablets to mobile.
  • Fast to Start: It's a tool for rapid development with best practices
  • Style Agnostic: It provides the most basic, beautiful styles, but is meant to be overwritten.
